World's First Online Crime Reporting System

Online crime reporting system came up with the solution to replace the old manual complaint registration system. The victims can file the case and submit the evidence in the website. Provides features like anonymous tip, alert systems, case status, crime notice board, cyber-crime complaint registration, GPS tracking system, police contact details with the station address, and provide the safe storage for storing evidence and files and maintain a backup of every single data.
